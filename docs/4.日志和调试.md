
# 日志和调试

## 日志系统

etlpy使用了python标准的logging模块作为日志输出，因此可以按照需求将其重定向到任意位置和格式，并设定输出等级。

合理的日志很重要。你可以设置不同的日志等级:

- >4: 对异常打印完整的信息堆栈
- >5: 对每条数据的处理过程给出完整信息
